# Data Dictionary

| Column | Description |
|---|---|
| Month | First day of the month |
| Month_Name | Display month label |
| Revenue_Budget | Planned monthly revenue |
| Revenue_Actual | Actual monthly revenue |
| COGS_Budget | Planned cost of goods sold |
| COGS_Actual | Actual cost of goods sold |
| Salaries_Budget | Planned salary expense |
| Salaries_Actual | Actual salary expense |
| Marketing_Budget | Planned marketing expense |
| Marketing_Actual | Actual marketing expense |
| Operations_Budget | Planned operations expense |
| Operations_Actual | Actual operations expense |
| Admin_Budget | Planned admin/G&A expense |
| Admin_Actual | Actual admin/G&A expense |
| Driver_Notes | Business explanation for monthly performance |
| Revenue_Variance | Actual revenue minus budget revenue |
| Revenue_Variance_Pct | Revenue variance divided by budget revenue |
| Gross_Profit_Budget | Budget revenue minus budget COGS |
| Gross_Profit_Actual | Actual revenue minus actual COGS |
| Total_Opex_Budget | Budget salaries + marketing + operations + admin |
| Total_Opex_Actual | Actual salaries + marketing + operations + admin |
| Operating_Profit_Budget | Gross profit budget minus total opex budget |
| Operating_Profit_Actual | Gross profit actual minus total opex actual |
| Profit_Variance | Actual operating profit minus budget operating profit |
| Operating_Margin_Actual | Actual operating profit divided by actual revenue |
| Revenue_Growth_Actual | Month-over-month actual revenue growth |
