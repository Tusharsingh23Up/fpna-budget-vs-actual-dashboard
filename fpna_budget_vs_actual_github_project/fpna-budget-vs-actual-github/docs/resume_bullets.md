# Resume Bullets

## Main Bullet

Built an FP&A dashboard using Excel and Python to analyze FY2025 budget vs actual variance, forecast revenue and operating profit, track profitability KPIs, and recommend cost-control actions for management reporting.

## Alternative Bullets

- Created a financial model with monthly revenue, COGS, opex, operating profit, variance analysis, and six-month scenario forecast.
- Designed an executive dashboard to monitor revenue variance, operating margin, profit variance, and controllable expense categories.
- Automated KPI calculations using Python and generated visual summaries for revenue trend, profit variance, and opex analysis.
- Converted raw budget and actual data into management insights, identifying June and September as key months for profitability review.
